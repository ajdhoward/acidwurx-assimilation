## 📡 Communication Protocol

### AI ↔ AI (Via Git)
1. **Read**: Check `tasks/backlog.md` for assigned work
2. **Write**: Update `agents/<name>.md` with progress every 30 min
3. **Blocker**: Create `prompts/external-help-<timestamp>.md`
4. **Complete**: Move task to `tasks/completed.md`

### AI ↔ Human
- **Daily Brief**: `reports/daily-brief-YYYY-MM-DD.md` (9 AM)
- **Blockers**: `reports/blockers.md` (immediate)
- **Prompts Ready**: `prompts/` directory (copy-paste to web AI)

### Human ↔ AI
1. Review `reports/daily-brief.md`
2. Copy prompt from `prompts/` → Web AI (Claude.ai/Gemini Web) → Click Send
3. Paste response → `prompts/<name>-response.md`
4. AI reads response, continues work
